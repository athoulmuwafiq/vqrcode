## Project setup
```
git clone https://github.com/viandwi24/vqrcode
cd vqrcode
npm install
npm  run serve
```

## What is used ?
* Vue.js
* Bootstrap, Jquery, Popper.js
* [fengyuanchen/vue-qrcode](https://github.com/fengyuanchen/vue-qrcode)
* [gruhn/vue-qrcode-reader](https://github.com/gruhn/vue-qrcode-reader)
